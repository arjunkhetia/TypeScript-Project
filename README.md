TypeScript is a language for application-scale JavaScript. TypeScript adds optional types to JavaScript that support tools for large-scale JavaScript applications for any browser, for any host, on any OS. TypeScript compiles to readable, standards-based JavaScript. The quickest way to get start with TypeScript is just clone the project:

```bash
$ git clone https://github.com/arjunkhetia/TypeScript-Project.git
```

Install dependencies:

```bash
$ npm install
```

Compile all TypeScript (.ts) files and produce JavaScript (.js) files to run:

```bash
$ npm run tsc
```

Execute translated JavaScript (.js) file:

```bash
$ npm start
```
